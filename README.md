## [svm](svm)
サポートベクターマシン(svm)を使える．svmにはマージン内への侵入を許可しないハードsvmと分離不可能な問題のさい学習の際に分類不可となるマージン内にあるデータの存在を認めるソフトsvmがあるが，ハードなsvmのみを実装したものはない．
許容の度合いはパラメータCの大きさに依存するため，大きくすればハードsvmに近づく．
